FUNDAMENTALS OF COMPUTER ENGINEERING - A COMPREHENSIVE INTRODUCTION TO THE CORE CONCEPTS AND PRINCIPLES OF COMPUTER ENGINEERING.

Subject Description

Fundamentals of Computer Engineering (FCE) is an introductory course that provides students with a solid foundation in the basic concepts of computer engineering. This subject bridges the gap between theoretical knowledge and practical application.

By the end of this course, students will have a comprehensive understanding of how computer systems are designed, built, and programmed, preparing them for more advanced subjects in the degree program.

Learning Objectives

  • Understand the fundamental concepts of computer architecture and organization.
  • Apply number systems and binary arithmetic in computer-related calculations.
  • Lear basic software development concepts and tools.
  • Comprehend how software interacts with hardware at the machine level.
  • Learn the current and past world of computer engineering in general.
