# Practical Work 2 - Personal Website

## Description

This project is a personal portfolio website created for the Fundamentals of The Computer Engineering course.

## Structure

```
docs/
├── css/style.css
├── images/
├── public/
│   ├── about.html
│   ├── contact.html
│   ├── net.html
│   ├── topic.html
│   ├── degree.html
│   └── fce.html
└── index.html
```

## Features

- **Brutalist Design**: Industrial aesthetic with monospace fonts and bold typography
- **Fully Responsive**: Works on desktop and mobile devices
- **Single CSS File**: All pages share a common stylesheet
- **No JavaScript Dependencies**: Pure HTML/CSS (minimal JS for decorative effects)

## Pages

| Page | Description |
|------|-------------|
| index.html   | Main landing page with site overview         |
| about.html   | Personal information, CV, and social links   |
| contact.html | Contact form (frontend only)                 |
| net.html     | Links to partner websites                    |
| topic.html   | Information on a chosen topic                |
| degree.html  | Degree structure and subjects                |

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Space Mono, Chakra Petch)

## Problems Encountered

- Ensuring consistent styling across all pages with a single CSS file
- Getting the striped background to work at all becayse of the z-index
- Ensuring consistent button styles and animations


## Conclusions

This project demonstrates proficiency in HTML and CSS fundamentals needed for basic web apps and web development in general, including external stylesheets, basic javascript, and responsive design principles, it also shows the non-necessity of common modern web frameworks that slow execution and increase page size.
